Снятие ограничений и разблокировка

Вредоносное ПО часто отключает системные утилиты управления. Данный модуль позволяет восстановить работоспособность заблокированных функций ОС в один клик.

Разблокировать Диспетчер задач

Восстанавливает заблокированный вирусом Task Manager (удаляет параметр DisableTaskMgr в реестре)

Разблокировать Редактор реестра

Восстанавливает доступ к regedit.exe (удаляет параметр DisableRegistryTools)

Разблокировать Командную строку

Убирает запрет администратора на запуск консоли командной строки (cmd.exe)

Сбросить файл Hosts

Очищает файл hosts от вирусных редиректов и восстанавливает его оригинальный чистый вид

Убрать запрет запуска ПО (DisallowRun)

Снимает блокировки с запуска защитного ПО и сторонних утилит из веток Explorer

Восстановить клавиатуру (Scancode Map)

Сбрасывает вредоносные переназначения клавиш в реестре, восстанавливая кнопки к работе

Встроенные утилиты обслуживания

Очистить временные папки системы

Удаляет временные кэши браузеров, папки Temp и логи установщиков для очистки свободного места.

Очистить сетевой кэш DNS

Применяет команду ipconfig /flushdns для восстановления нормального сетевого подключения.

Восстановление ассоциаций системных файлов

Исправляет запуск .exe, .bat, .lnk файлов, если вирус нарушил их нормальный запуск в Windows.

Проверить целостность файлов (SFC)

Запускает системное средство sfc /scannow в новом окне для восстановления системных библиотек.
